# Privacy Policy — CopyTitle

## Overview

CopyTitle is a Chrome extension that allows users to copy the current page’s title and URL in different formats (plain text, Markdown, and HTML).

This privacy policy explains what data the extension accesses and how it is handled.

## Data Collection

CopyTitle does **not collect, transmit, or sell any personal data**.

The extension does not send any information to external servers.

## Data Accessed by the Extension

To function, CopyTitle temporarily accesses:

* The active tab’s **URL**
* The active tab’s **page title**

This information is used only when the user clicks a button or triggers a shortcut.

## Data Storage

CopyTitle stores only one local preference using Chrome storage:

* `lastAction`: remembers the last used copy format (e.g., Markdown, URL, HTML)

This data is:

* stored locally on the user’s device
* never transmitted anywhere
* can be cleared at any time by removing the extension

## Clipboard Access

The extension writes user-selected content (URL/title combinations) to the clipboard.

It does not read clipboard content.

## Permissions Usage

CopyTitle requests the following Chrome permissions:

* `activeTab`: to access the current tab’s URL and title when the user interacts with the extension
* `clipboardWrite`: to copy content to the clipboard
* `storage`: to save the last used action locally

No other permissions are used.

## Third-Party Services

CopyTitle does **not** use:

* analytics tools
* tracking tools
* remote code
* external APIs
* advertising networks

## Data Sharing

No data is shared with any third party.

## Security

All operations are performed locally inside the browser. No network requests are made.

## Changes to This Policy

This privacy policy may be updated in the future. Any changes will be reflected in this document.
