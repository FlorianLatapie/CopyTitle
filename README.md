# CopyTitle

## How to use

1. Clone this repository **or** download the zip file and unzip it.
2. Open `chrome://extensions` in your chromium based browser (Chrome, Edge ...).
3. Enable `Developer mode`.
4. Load the folder using the `Load unpacked` button.

## If you are unable to load the extension

Use the bookmarklet version:

1. Open the raw file [bookmarklet.js](./bookmarklet.js).
2. Save the page as a bookmark in your browser.
   1. Click on the star icon in the address bar (or press `Ctrl + D` / `Cmd + D`).
   2. Click on `More options`.
   3. In the `URL` field, replace the URL with the content of the raw file.
   4. Save the bookmark.
